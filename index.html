Here you will learn more about functions and how to use them.

Use these in the devtools toget different responses

  • "console.log(jumpOverFence(fox))" should give "Jumped Over The Lazy Dog Over The Fence"
  • "console.log(isEvenOrOdd(10))" should be true
  • "console.log(isEvenOrOdd(5))" should be false
  • "console.log(celsiusToFahrenheit(30))" should give 86
  • "console.log(celsiusToFahrenheit(0))" should give 32
  • "console.log(findMaxNumber([1, 2, 3, 4, 5]))" should give 5
  • "sayHelloWorld()" should send "hello world" after 2 seconds
  • "console.log(squareNumber(4))" should give 16
  • "console.log(repeatString("Hello"))" should give "hellohellohello"
  • "console.log(scopingData())" should say "hello world"
  • "console.log(reverseString("Hello"))" should reverse the word given
  • "console.log(randomNumber())" should give a random number between 1 and 10
  • "console.log(findLargestNumber([3, 6, 2, 8, 4]))" should give 8
  • "console.log(removeDuplicates([1, 2, 2, 3, 4, 4, 5]))" should make it so there are no repeat numbers
  • "console.log(filterEvenNumbers([1, 2, 3, 4, 5]))" should only give 2 and 4
  • "console.log(filterArray([1, 2, 3, 4, 5], 2))" make the list go "3, 4, 5, 1, 2"
  • "console.log(rotateArray([1, 2, 3, 4, 5], 2))" make the list go "4, 5, 1, 2, 3"
